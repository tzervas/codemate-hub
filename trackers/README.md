Trackers

This directory contains project-level tracking documents, a project plan, technical spec, and per-task trackers to manage the work for this repository.

Structure:
- `OVERVIEW.md` - concise repo intent & architecture summary
- `PLAN.md` - milestones, timeline, and milestones
- `SPEC.md` - technical specification and acceptance
- `tasks/` - per-area task trackers (each file contains subtasks, estimates, acceptance criteria)

How to use:
- Update the relevant file in `tasks/` when you start a subtask.
- Use `PLAN.md` to track milestones and deliveries.
