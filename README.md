# Dockerized Agentic Coding Assistant

## Quick Start
1. Set env: export PASSWORD=your_pass
2. Build: ./scripts/build.sh
3. Deploy: ./scripts/deploy.sh
4. Access: Langflow (7860), code-server (8080)

## Structure
- src/: Core Python code
- scripts/: Automation
- zephyr/: Enclaves
- insights/: Bootstraps

For details, see deploy scripts.
